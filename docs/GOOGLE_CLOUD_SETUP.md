# Google Cloud 설정 상세 가이드 (바로가기 링크 포함)

만약 메뉴를 찾기 어렵다면, **아래 파란색 링크를 클릭**하면 해당 페이지로 바로 이동합니다!

---

## 1단계: 프로젝트 만들기

1.  👉 **[프로젝트 생성 페이지 바로가기](https://console.cloud.google.com/projectcreate)** 클릭
2.  **프로젝트 이름**: `Currency Calculator` (또는 원하는 이름) 입력
3.  **조직**: '조직 없음' 상태 그대로 둠 (개인 계정인 경우)
4.  하단의 **[만들기]** 파란색 버튼 클릭
    *   *잠시 기다리면(약 10~30초) 우측 상단에 "프로젝트 만들기 완료" 알림이 뜹니다.*
5.  👉 **[대시보드로 이동](https://console.cloud.google.com/home/dashboard)** 클릭 후, 상단 파란색 바에서 방금 만든 프로젝트(`Currency Calculator`)가 선택되어 있는지 확인하세요.

---

## 2단계: OAuth 동의 화면 만들기

가장 먼저 "이 앱은 누가 사용할 것인가?"를 설정해야 합니다.

1.  👉 **[동의 화면 설정 페이지 바로가기](https://console.cloud.google.com/apis/credentials/consent)** 클릭
2.  **User Type (사용자 유형)**:
    *   **외부 (External)** 선택 (체크박스)
    *   하단 **[만들기]** 버튼 클릭
3.  **앱 정보 입력** (필수 항목만 입력하면 됩니다):
    *   **앱 이름**: `환율 계산기`
    *   **사용자 지원 이메일**: 드롭다운을 눌러 본인 이메일 선택
    *   (아래로 스크롤) **개발자 연락처 정보**: 본인 이메일 직접 입력
4.  맨 아래 **[저장 후 계속]** 버튼 클릭
5.  **범위 (Scopes)** 페이지:
    *   아무것도 건드리지 말고 맨 아래 **[저장 후 계속]** 클릭
6.  **테스트 사용자 (Test Users)** 페이지:
    *   중간에 있는 **[+ ADD USERS]** 버튼 클릭
    *   본인의 구글 이메일 주소 입력 후 [추가]
    *   맨 아래 **[저장 후 계속]** 클릭
7.  **요약** 페이지:
    *   맨 아래 **[대시보드로 돌아가기]** 클릭

---

## 3단계: OAuth 클라이언트 ID 발급 (로그인 기능용)

이제 로그인 버튼에 들어갈 "열쇠(ID)"를 만듭니다.

1.  👉 **[사용자 인증 정보 만들기 페이지 바로가기](https://console.cloud.google.com/apis/credentials)** 클릭
2.  화면 상단에 있는 **[+ 사용자 인증 정보 만들기]** 글자를 클릭 (플러스 아이콘)
3.  나오는 메뉴 중 **OAuth 클라이언트 ID** 클릭
4.  **애플리케이션 유형**: 드롭다운에서 **웹 애플리케이션** 선택
5.  **이름**: `웹 클라이언트 1` (변경 안 해도 됨)
6.  (중요) **승인된 자바스크립트 원본** 항목 찾기:
    *   **[+ URI 추가]** 버튼 클릭
    *   아래 주소들을 하나씩 입력해 주세요 (개발할 때와 배포할 때 필요):
        *   `http://127.0.0.1:5500`
        *   `http://localhost:5500`
        *   `https://YOUR_GITHUB_USERNAME.github.io` (나중에 실제 깃허브 주소로 수정 필요)
7.  맨 아래 **[만들기]** 파란색 버튼 클릭
8.  ✨ **팝업창이 뜨면 성공!**
    *   **"클라이언트 ID"**라고 적힌 긴 문자열을 복사해서 메모장에 저장해 두세요.
    *   (예: `123456789-abcdefg...apps.googleusercontent.com`)
    *   *클라이언트 보안 비밀번호는 필요 없습니다.*

## 4단계: Gemini API 키 발급 (OCR 기능용)

AI에게 영수증을 읽어달라고 부탁할 "출입증"을 만듭니다.

1.  👉 **[Google AI Studio 바로가기](https://aistudio.google.com/app/apikey)** 클릭
2.  파란색 **[Create API key]** 버튼 클릭
3.  **Select Project**: 아까 만든 `Currency Calculator` 프로젝트를 선택하거나, "Create API key in new project"를 선택해도 됩니다.
4.  **[Create API key in existing project]** 등을 눌러 생성
5.  ✨ **생성된 키 복사**: `AIzaSy...` 로 시작하는 긴 문자열을 복사해서 메모장에 저장하세요.
